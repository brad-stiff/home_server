# Welcome to the backend

## Get started

1. install
   ```bash
   npm install express mysql2 cors body-parser
   ```
2. start the app
   ```bash
   npm run dev
   ```

## MySQL

1. login
   ```bash
   /usr/local/mysql/bin/mysql -u root -p
   ```
2.mysql commands
   ```bash
   show databases;
   ```
